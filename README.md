# Anime Quiz Application

A professional, interactive quiz application showcasing anime characters and series. Test your anime knowledge with an engaging, responsive user interface.

Live Demo(Github Pages): https://kimi1407.github.io/ANIME-QUIZ/

## Features

- **Two Quiz Modes**: Character identification and anime series recognition
- **User Tracking**: Personalized experience with user name input
- **Progress Tracking**: Real-time question counter and score display
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Enhanced user experience with CSS transitions
- **Result Summary**: Detailed performance feedback after quiz completion

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs entirely in the browser

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/Kimi1407/ANIME-QUIZ.git
cd ANIME-QUIZ
```

2. Open in your browser:
   - Double-click `index.html` or
   - Use a local server: `python -m http.server 8000` then visit `http://localhost:8000`

### Online Demo
Visit: [https://kimi1407.github.io/ANIME-QUIZ/](https://kimi1407.github.io/ANIME-QUIZ/)

## Quiz Content

### Character Quiz
Identify anime characters from their images. Features characters from popular series including:
- Fire Force
- Dr. Stone
- Attack on Titan
- And more...

### Anime Series Quiz
Guess anime titles from key visual scenes. Includes recognition of:
- Corpse Party
- Dandadan
- Dr. Stone
- The Promised Neverland
- A Silent Voice

## Performance Metrics

- Quiz completion time: ~3-5 minutes per quiz
- 5 questions per quiz mode
- Final score and accuracy percentage displayed
